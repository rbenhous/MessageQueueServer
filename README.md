# MessageQueueServer
TabNine assignment - server for managing queues of messages.

## Installation
1. Install npm required packages
```bash
npm install
```
2. Create .env file in the root directory and add the following environment variables:
```bash
PORT = 3000
MAX_QUEUE_SIZE = 100
DEFAULT_TIMEOUT = 10
```

## Running the app
```bash
# development
npm run start
```
